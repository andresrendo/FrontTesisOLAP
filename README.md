# FrontendTesis

Interfaz React para visualizar resultados comparativos entre PostgreSQL y MonetDB en consultas analíticas OLAP.

## Estructura
- Página principal con resumen y objetivos
- Tabla de resultados
- Gráfica comparativa

## Instalación
1. Instala dependencias:
   ```bash
   npm install
   ```
2. Inicia el proyecto:
   ```bash
   npm start
   ```
